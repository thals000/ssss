4단계: 구상 설계 (심상)
핵심 원칙:
  • 음식을 먼저 정하고 재료를 가져와라
  • "이쁜 그림 아무거나" 금지 (요리 재료 없는 것과 같음)
  • 정해놓은 구성설계가 있어야 무슨 포즈를 어떻게 그릴지 알 수 있음
상황 만들기:
  • 키워드를 잡고 상상하기
  • 자세, 표정, 위치, 보폭, 주변의 요소 등등 정하기
  • 구상설계, 스토리텔링을 먼저 다 해야 함

5단계: 러프 과정 (선 그리기)
단계별 진행:
  • 1. 도형화 - 위치, 비율, 모양을 정확하게 도형화
  • 2. 외곽선 (CSI) - 오퍼시티 줄이고 크게크게
  • 3. 내부 연결 (CSI) - 안에 또 이어주기
  • 4. 세부선 (631) - 개얇은 선, 손 안 떼고 이쁘게
  • 5. 실루엣 완성 - 여기까지가 실루엣
  • 6. 형태 디테일 - 겉에 모양, 외곽 디테일
주의사항: 안에만 들어가야 함, 밖으로 나가지 말기
